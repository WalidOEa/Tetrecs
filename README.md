# Tetrecs
## About
A short, fun game based off the hit game, Tetris.

## Requirements
- JDK 17+
- Maven 3.2+

## How to Run
1. In cmd or terminal, navigate to the root directory of Tetrecs, containing file Pom.xml
2. Run command,

    ```mvn javafx:run```
